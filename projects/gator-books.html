GATOR BOOKS

DEC 2022

Project Description

Gator Books is a console application that allows users to search through a vast database of books and utilize efficient sorting algorithms like Timsort and Radix sort to organize data. By leveraging this application, users can conveniently compare the execution times of these two sorting algorithms.

The database is sourced from Kaggle. This database provides a comprehensive collection of high-quality book records for users to search through and analyze.
