Jan - Apr 2023

Project Description

This comprehensive portfolio comprises six distinct web pages, each highlighting the progression of my skills in the realm of web development.

The initial two assignments focused on mastering the fundamentals of HTML, CSS, and Javascript, with an emphasis on building solid foundations without relying on external libraries.

As I delved deeper into the course, I utilized more powerful tools like Bootstrap and jQuery in assignments 3-4, enabling me to create more dynamic and visually appealing web experiences.

In assignment 5, I began some server-side programming, acquainting myself with PHP while seamlessly integrating it with the previously acquired tools.

In assignments 6-7, I combined a multitude of technologies. These final projects showcased my proficiency in Javascript, HTML/CSS, AJAX, jQuery, PHP, and SQL to create a responsive and user friendly webpage.
